# Ask Uni Mate

Welcome to Ask Uni Mate!

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the port specified in your console) with your browser to see the result.

## Build for Production

To build the app for production, run:

```bash
npm run build
```

## Deployment

The project contains a `vercel.json` and is ready to be deployed on Vercel.
